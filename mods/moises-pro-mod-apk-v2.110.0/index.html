4. Tips Penggunaan Maksimal

4.1. Menyiapkan File Audio Berkualitas

Untuk hasil optimal, gunakan file WAV 44.1 kHz 16-bit atau FLAC. Hindari MP3 berbitrate rendah karena artefak kompresi bisa menurunkan akurasi pemisahan. Jika hanya tersedia MP3, pilih bitrate minimal 192 kbps. Panjang lagu maksimal 10 menit untuk mode gratis, namun MOD menghapus batasan ini. Pastikan lagu stereo agar AI bisa memanfaatkan informasi spatial.

4.2. Pilih Mode Pemisahan yang Tepat

Mode "5-Stem" ideal untuk remix atau cover penuh. Jika hanya ingin membuat minus one, gunakan "Vokal + Instrumental" untuk hemat waktu. Mode "Drum + Bass + Other" berguna untuk beat maker yang hanya perlu groove. Selalu gunakan preview 30 detik untuk mengecek apakah stem sesuai kebutuhan sebelum memproses full.

4.3. Manfaatkan Keyboard Shortcut

Pada layar proyek, tekan R untuk record, Spasi untuk play/pause, dan L untuk loop. Tombol panah kiri/kanan bergeser 5 detik, sambil Ctrl + panah geser 30 detik. Shortcut ini mempercepat workflow saat editing. Tersedia juga gesture swipe dua jari untuk zoom timeline di layar kecil.

4.4. Backup & Arsitektur File

Buat folder bernama "Moises Projects" di cloud pribadi. Di dalamnya, buat subfolder berdasarkan tanggal atau nama band. Simpan file master, stem terpisah, serta file chord PDF dalam satu paket. Dengan demikian, Anda bisa restore proyek kapan saja, bahkan jika akun Moises bermasalah. Fitur "Export Bundle" di MOD akan otomatis mengarsipkan seluruh file menjadi ZIP.
