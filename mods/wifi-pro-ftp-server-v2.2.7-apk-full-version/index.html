4. Tips & Trik

4.1. Atur Port Forwarding untuk Akses Jarak Jauh

Anda bisa mengakses server dari luar rumah dengan port forwarding di router. Gunakan high-port (di atas 1024) untuk menghindari scanner bot. Pastikan juga mengaktifkan IP whitelist agar hanya IP tertentu yang boleh terhubung.

4.2. Sinkronisasi Otomatis dengan Folder Terpilih

Gunakan client seperti FolderSync untuk menjaga agar foto atau video terbaru langsung terbackup ke PC. Aktifkan opsi "Mirror" agar file yang dihapus di ponsel juga otomatis terhapus di server, menjaga konsistensi ruang penyimpanan.

4.3. Monitoring Real-Time lewat Notifikasi

Di pengaturan, aktifkan "Speed in Notification". Maka kecepatan transfer akan muncul persistent di bilah notifikasi, memudahkan Anda mengetahui apakah koneksi sedang stabil atau mengalami penurunan.

4.4. Mengatasi Masalah Windows Explorer

Jika Windows Explorer sering putus saat mengirim file besar, ubah mode transfer ke "Binary" di opsi lanjutan. Explorer default menggunakan ASCII yang bisa menyebabkan kesalahan EOF.
