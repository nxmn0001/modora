4.1. Gunakan Shortcut Keyboard Bluetooth

Hubungkan keyboard QWERTY bluetooth agar shortcut seperti Ctrl+Z, Ctrl+C, dan Space untuk play/stop lebih cepat. Daftar lengkap shortcut ada di Settings > Shortcuts.

4.2. Manfaatkan Per-Project Samples

Simpan sample khusus di folder project agar tidak bercampur dengan library lain. Ini membuat backup dan sharing project lebih ringkas.

4.3. Rutin Freeze Track

Kalau track synth sudah selesai diedit, freeze agar mengurangi beban CPU. Tekan ikon gembok di mixer. File audio hasil freeze otomatis tersimpan.

4.4. Ambil Sample dari Lingkungan

Manfaatkan mic built-in untuk merekam suara lingkungan: desau daun, kicau burung, atau derap jalan kaki. Gunakan filter high-pass untuk hasil natural.
