Untuk memaksimalkan pengalaman, terapkan strategi berikut:

Anda bisa menentukan jenis file mana saja yang ingin di-backup. Aktifkan centangan hanya pada foto & video, lalu sisipkan ekstensi .pdf jika perlu dokumen. Cara ini mempercepat proses pemindaian.

Manfaatkan fitur Color Tag untuk menandai folder prioritas. Misalnya, merah untuk proyek aktif, kuning untuk arsip, hijau untuk file pribadi. Pencarian berdasarkan tag sangat cepat.
