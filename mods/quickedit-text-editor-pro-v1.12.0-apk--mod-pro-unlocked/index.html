4. Tips & Trik

4.1. Membuka File Besar Tanpa Lag

Matikan opsi "Syntax Highlighting" sementara bila file melebihi 5 MB. Setelah selesai editing aktifkan kembali untuk melihat warna kode.

4.2. Pintasan Keyboard Eksternal

Sambungkan keyboard Bluetooth/USB OTG, lalu gunakan Ctrl+S untuk save, Ctrl+F untuk find, dan Ctrl+Z untuk undo agar produktivitas meningkat.

4.3. Sinkronisasi Otomatis

Pastikan akun cloud terverifikasi dan kuota penyimpanan masih tersedia. Aktifkan "Auto-sync on Wi-Fi only" agar kuota data seluler tak boros.

4.4. Backup Lokal

Gunakan fitur "Export to ZIP" untuk mem-backup seluruh proyek dalam satu arsip. Simpan ZIP tersebut di folder Download agar mudah dibagikan ke PC.
