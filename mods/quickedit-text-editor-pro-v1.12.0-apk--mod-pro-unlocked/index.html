4. Tips Penggunaan

4.1. Shortcut Keyboard OTG

Jika kamu menghubungkan keyboard fisik via OTG, gunakan Ctrl+S untuk save, Ctrl+F untuk find, Ctrl+Z/Y untuk undo/redo, dan Ctrl+D untuk duplikat baris. Shortcut sama seperti desktop, sehingga migrasi lebih mudah.

4.2. Gunakan Macro untuk Tugas Berulang

Rekam macro untuk membersihkan baris kosong, menambahkan header, atau konversi case. Simpan macro ke dalam koleksi, lalu jalankan dengan satu ketukan di project berikutnya.

4.3. Atur Auto-Backup

Aktifkan auto-backup ke folder cloud favoritmu. Atur interval 10 menit agar perubahan tersimpan otomatis, meminimalkan risiko kehilangan data saat baterai tiba-tiba habis.

4.4. Optimasi Tema Gelap

Pilih tema "Monokai Pro" untuk pengkodean malam hari, atau buat palet warnamu sendiri. Kurangi kecerahan latar agar baterai lebih hemat, terutama pada layar AMOLED.
